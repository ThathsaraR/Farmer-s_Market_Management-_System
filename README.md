# Farmer’s Market Management System

A web-based system developed to manage vendor registration, stall reservations, product listings, and customer orders for a weekly farmer’s market.

## Project Overview
The system replaces manual management with an online platform where:
- Farmers register as vendors and manage their profiles
- Vendors reserve stalls and list products for market days
- Customers browse products and place pickup reservations
- Administrators manage vendors, stalls, events, and reports

## Key Features
- Vendor registration and profile management
- Stall reservation and market event scheduling
- Product listing with price and availability
- Customer product reservations for pickup
- Vendor–customer messaging system
- Admin dashboard for monitoring and reports

## System Implementation
- Web-based user interface with consistent navigation
- Full CRUD operations (Create, Read, Update, Delete)
- Data validation and user-friendly interactions

## Database Implementation
- Primary keys and foreign key relationships enforced
- Referential integrity maintained
- Tested using sample data

## Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL
